# MDJ St-Sauveur / Piedmont

Site statique multi-page pour la Maison des jeunes St-Sauveur / Piedmont.

## Prévisualiser

Ouvrir `index.html` directement dans un navigateur.

Optionnel, avec Node.js:

```powershell
node serve-local.mjs
```

Puis visiter:

```text
http://127.0.0.1:4173
```

## Pages

- `index.html` - Accueil
- `ta-mdj.html` - Ta MDJ
- `aide-aux-devoirs.html` - Aide aux devoirs
- `commanditaires.html` - Commanditaires
- `a-propos.html` - À propos
- `contact.html` - Contact

## Assets à remplacer plus tard

- `public/images/logo-mdj.png` doit être remplacé par le fichier officiel fourni par le client si une version plus propre est remise.
- Les photos Facebook réelles peuvent être ajoutées dans les blocs visuels une fois leur utilisation validée.
- Les logos des commanditaires doivent être ajoutés seulement après réception des fichiers autorisés.
- Le formulaire de contact est visuel et prêt à connecter à un backend ou à un service de courriel.
